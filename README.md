## Running the project
1. Clone this repository
2. Run `npm install` on the project root directory to install dependencies
3. Set up environment variables
   - **LOCAL_STORAGE_KEY**: key used to encrypt locally stored data
   - **GOOGLE_WEB_CLIENT_ID**: linked client ID used for Google Sign In
   - **API_ROOT_URL**: the URL of the backend API. To see how to run and configure API go to: https://github.com/adferdv/analock-api
